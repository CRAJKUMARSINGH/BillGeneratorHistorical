# Word Files for Easy Editing

## Why Word Files?

Word (.docx) files are perfect for **last-minute edits by non-technical users** like:
- Semi-literates who can't edit HTML
- Clerks who need to make quick corrections
- Officers who want to review and modify before final submission

## Installation

```bash
pip install python-docx
```

## How It Works

Once installed, the batch processor will automatically generate:
- ✅ `first_page.html` → `first_page.docx`
- ✅ `deviation_statement.html` → `deviation_statement.docx`
- ✅ `extra_items.html` → `extra_items.docx`
- ✅ All other documents

## Benefits

1. **Easy Editing** - Anyone can edit in Microsoft Word
2. **No HTML Knowledge** - Just type and save
3. **Preserve Formatting** - Tables, borders, fonts all maintained
4. **Quick Corrections** - Fix typos, update amounts, change dates
5. **Print Ready** - Print directly from Word

## Usage

1. Run batch processor: `RUN_BATCH_ALL.bat`
2. Find Word files in output folder alongside HTML files
3. Open in Microsoft Word
4. Make your edits
5. Save and print!

## Perfect For

- Last-minute amount corrections
- Date changes
- Name corrections
- Remark additions
- Quick reviews before submission

## Note

The Word files have the **same formatting** as HTML/PDF but are **fully editable**!
